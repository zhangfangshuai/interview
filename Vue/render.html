render()

template 和 render()什么关系

  1. template是Vue框架设定的一种语法糖，并不是真正的dom节点
  2. templte里的内容需要通过render()函数去创建对应的真实dom，再生成虚拟dom，最后转化为真实dom
  3. 流程是： template => render() => 虚拟dom => 转化为真实dom

render(h)里的h代表什么

  1. h => 原声JS中的 createElement()
